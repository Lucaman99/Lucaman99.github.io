Pick some $a$ such that $f(a) \ \neq \ g(a)$. Let us define $|f(a) \ - \ f(g)| \ = \ 2\epsilon$, which will be greater than $0$. Now, since $f$ and $g$ are both continuous, it follows that if we consider the number $\epsilon$, we can choose $\delta_f$ and $\delta_g$, respectively, such that: $$|x \ - \ a| \ < \ \delta_f \ \Rightarrow \ |f(x) \ - \ f(a)| \ < \ \epsilon$$ $$|x \ - \ a| \ < \ \delta_g \ \Rightarrow \ |g(x) \ - \ g(a)| \ < \ \epsilon$$ Let $\delta \ = \ \text{min} \{\delta_f, \ \delta_g\}$. By the triangle inequality, we have: $$|f(x) \ - \ f(a)| \ + \ |f(x) \ - \ g(x)| \ + \ |g(x) \ - \ g(a)| \ \geq \ |f(a) \ - \ g(a)| \ = \ 2\epsilon$$ for some $x$. This gives us: $$|f(x) \ - \ g(x)| \ \geq \ 2\epsilon \ - \ |f(x) \ - \ f(a)| \ - \ |g(x) \ - \ g(a)|$$ Thus, for all $x$ such that $|x \ - \ a| \ < \ \delta$, we have: $$|f(x) \ - \ g(x)| \ \geq \ 2\epsilon \ - \ |f(x) \ - \ f(a)| \ - \ |g(x) \ - \ g(a)| \ > \ 2\epsilon \ - \ \epsilon \ - \ \epsilon \ = \ 0$$ $$\Rightarrow \ |f(x) \ - \ g(x)| \ > \ 0$$ This concludes the proof of the lemma, as the above line implies that $f(x) \ \neq \ g(x)$ for all $x$ such that $|x \ - \ a| \ > \ \delta$, for some $\delta$.

Assume that there exists an irrational number $r$ such that $f(r) \ \neq \ g(r)$. By the previous lemma, it follows that there exists some $\delta \ > \ 0$ such that if $|x \ - \ r| \ < \ \delta$, then $f(x) \ \neq \ g(x)$. All such $x$ will be contained in the interval $L \ = \ (r \ - \ \delta, \ r \ + \ \delta)$. Since there exists a rational number between any two real numbers, it follows that there is some rational $s \ \in \ L$, which would imply that $f(s) \ \neq \ g(s)$. This contradicts the original assumption that the functions are equal over the rationals. Thus, there must exists no such $r$, and the functions are equal over the reals.
Assume that $f(x)$ is not a constant function. This implies that there exists $a$ and $b$ such that $f(a) \ \neq \ f(b)$. By mean value theorem, for some $r$ between $f(a)$ and $f(b)$, there exists some $c$ such that $f(c) \ = \ r$. Since there is an irrational number between every pair of rational numbers, there will be some $f(c)$ equal to an irrational $r$. This contradicts our initial assumption, thus $f$ must be constant.
