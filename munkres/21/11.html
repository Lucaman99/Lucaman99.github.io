Part A

We start off by proving a fairly simple statement: if $(s_n)$ is a bounded sequence of real numbers, and $s_n \ \leq \ s_{n \ + \ 1} \ \forall n$, then the sequence $(s_n)$ converges. In order for this sequence to converge to some number $s$, every neighbourhood of $s$ must contain $s_n \ \forall \ n \ > \ N$, for some value of $N$. If the sequence $(s_n)$ is bounded, then there must exist a lower bound and an upper bound on the set $(s_n)$. We will pick $s_a \ = \ \sup (s_n)$.

There are two cases that we can consider. The first is when an element of the sequence repeats itself an infinite number of times, which in that caase means that the series converges to the repeated number. This can be easily proven: if $s_n \ = \ s$ for $n \ > \ N$, then any open interval of $s$ contains $s_n$ for $n \ > \ N$, thus the sequence converges to $s$.

In addition, we can consider the case where the sequence does not repeat itself an infinite number of times. This means that for some $s_n$, we can always find $s_m$ with $m \ > \ n$ such that $s_n \ \neq \ s_m$. Since $s_{n \ + \ 1} \ \geq \ s_n$, it follows that $s_m \ > \ s_n$. We extend this, to write that for every $n$, there exists some $m$ such that:

$$s_n \ < \ s_m \ < \ s_a \ \ \ \text{for} \ n \ < \ m$$

And in addition, $s_n \ \leq \ s_{n \ + \ 1}$ for every $n$. It thus follows that our sequence must approach $s_a$. We assert that our sequence converges to $s_a$. We choose some open set $(s_a \ - \ \epsilon, \ s_a \ + \ \epsilon)$ around $s_a$. Now, since we can always find $s_m \ > \ s_n$, we can get arbitrarily close to $s_a$, with some element $s_N$. It follows from the fact that $s_n \ \leq \ s_{n \ + \ 1}$ that every $s_n$ with $n \ \geq \ N$ is is the open interval of $s_a$, thus proving our sequence converges to $s_a$. $\blacksquare$
